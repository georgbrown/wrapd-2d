The sample meshes provided in this directory were obtained from three
different sources. Below we identify the sources and, when different,
the original file names in parantheses.

1)  Yufeng Zhu, Robert Bridson, Danny Kaufman.
    Blended Cured Quasi-Newton for Distortion Optimization.
    ACM Transactions on Graphics, SIGGRAPH 2018, Vancouver.

    https://github.com/mike323zyf/BCQN

    Files:
    - bull.obj (bull_1.obj)
    - camel.obj (camel_2.obj)
    - dino.obj (dino_2.obj)
    - hilbert.obj
    - horse.obj
    - isis.obj (isis_1.obj)

2)  Ligang Liu, Chunyang Ye, Ruiqi Ni, Xiao-Ming Fu.
    Progressive Parameterizations.
    ACM Transactions on Graphics (SIGGRAPH) 37(4), 2018.

    Shuangming Chai, Xiao-Ming Fu, Xin Hu, Yang Yang, Ligang Liu.
    Sphere-based Cut Construction for Planar Parameterizations.
    Computer & Graphics (SMI 2018).

    http://staff.ustc.edu.cn/~fuxm/projects/ProgressivePara/dataset.html

    Files:
    - creature.obj (D1_00935 in the D1 dataset)
    - dog.obj (D2_02326_n09 in the D2 dataset)
    - snake.obj (D2_04530_n05 in the D2 dataset)

3)  Jian-Ping Su, Chunyang Ye, Ligang Liu, Xiao-Ming Fu.
    Efficient Bijective Parameterizations.
    ACM Transactions on Graphics (SIGGRAPH), 39(4), 2020.

    https://rec.ustc.edu.cn/share/fe925030-a145-11ea-9033-53f378dddcd3

    Files:
    - lucy.obj (fig8_1_input.obj)
